Reality Check Foundation v3

What this package adds:
- front-door welcome screen
- account/login shell (prototype only)
- profile intake screen
- next/back navigation buttons
- lighter blue radio text
- stronger app shell
- final decision page
- recommendation engine

Files included:
- app.py
- theme.py
- nav_ui.py
- welcome_ui.py
- profile_ui.py
- shared_ui.py
- recommendation_engine.py
- overview_ui.py
- final_decision_ui.py

Files you should keep from your current app:
- phase0_questions.py
- phase0_logic.py
- phase0_ui.py
- phase1_questions.py
- phase1_logic.py
- phase1_ui.py
- financial_model_ui.py
- logo.png

How to use:
1. Copy these files into your app folder.
2. Replace existing files where the names match.
3. Keep your current Phase 0 / Phase 1 / financial model files.
4. Run the app again.

Command:
cd ~/Desktop/reality_check_app
source .venv/bin/activate
streamlit run app.py
